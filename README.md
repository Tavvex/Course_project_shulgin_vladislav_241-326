# Сайт-портфолио художника Мошуровой Полины

Сайт-визитка художника с возможностью управления контентом через админ-панель.

## Возможности

- 📸 Портфолио с фильтрацией по категориям и годам
- ✍️ Система отзывов с модерацией
- 🎨 Управление работами через админ-панель
- 📁 Благотворительный раздел с загрузкой файлов
- 🔒 Защита от CSRF, XSS, брутфорса
- 📧 Email уведомления о подозрительной активности

## Технологии

- Python 3.11+
- Flask 2.3.3
- SQLite (разработка) / PostgreSQL (production)
- HTML/CSS/JavaScript

## Установка для разработки

```bash
# Клонировать репозиторий
git clone https://github.com/your-username/artist-site.git
cd artist-site

# Создать виртуальное окружение
python -m venv venv

# Активировать (Windows)
venv\Scripts\activate

# Установить зависимости
pip install -r requirements.txt

# Создать .env файл
cp .env.example .env

# Запустить приложение
python run.py
```
