# Сайт-портфолио художника Мошуровой Полины

Сайт-визитка художника с возможностью управления контентом через админ-панель.

## Возможности

- 📸 Портфолио с фильтрацией по категориям, статусу (продаётся/не продаётся) и сортировкой по году
- ❤️ Избранное для зарегистрированных пользователей
- ✍️ Система отзывов с модерацией
- 🎨 Управление работами через админ-панель
- 📁 Благотворительный раздел с загрузкой и скачиванием файлов
- 👤 Регистрация, вход, личный кабинет, восстановление пароля по email
- 🛠 Управление пользователями и отзывами через админ-панель

## Технологии

- Python 3.11+
- Flask, Flask-SQLAlchemy, Flask-Login
- SQLite (разработка) / PostgreSQL (production)
- Waitress (production-сервер)
- HTML/CSS/JavaScript

## Структура проекта

```
WebAplication/
├── app/
│   ├── __init__.py        # фабрика приложения (create_app)
│   ├── config.py           # конфигурация (пути, БД, загрузки)
│   ├── extensions.py       # инициализация db, login_manager
│   ├── models.py           # модели: User, Work, Review, Favorite, CharityFile
│   ├── mail.py              # отправка писем восстановления пароля
│   ├── utils.py             # работа с загружаемыми файлами
│   ├── routes/              # blueprints: public, auth, cabinet, admin
│   ├── static/              # css, js, изображения
│   ├── templates/           # Jinja2-шаблоны
│   ├── database/            # файл базы данных (создаётся автоматически)
│   └── logs/                 # логи приложения
├── run.py                     # точка входа: запуск приложения (waitress/Flask dev-сервер)
├── requirements.txt
├── .env.example
└── .gitignore
```

## Установка для разработки

```bash
# Клонировать репозиторий
git clone https://github.com/your-username/artist-site.git
cd artist-site

# Создать виртуальное окружение
python -m venv venv

# Активировать (Windows)
venv\Scripts\activate

# Установить зависимости
pip install -r requirements.txt

# Создать .env файл
cp .env.example .env

# Запустить приложение
python run.py
```

После запуска сайт будет доступен на http://127.0.0.1:5000.

## Администратор

При первом запуске, если в базе нет ни одного администратора, автоматически
создаётся учётная запись:

- Логин: `admin`
- Пароль: `CHANGE_THIS_PASSWORD_NOW!`

Панель администратора доступна по адресу `/admin/`. **Сразу после первого
входа смените пароль через личный кабинет.**
